ITとは？
IT:Information Technology、情報技術

OS
OS:Operating Sytstem システム全体を管理し、さまざまなアプリケーションソフトを動かすための最も基本的なソフトウェア
Windows Microsoftが提供しているOS
macOS (Mac OS X) Appleが提供しているOS
UNIX OSに一種で、最も古いOSの一つ。
Linux リーナス・トーバルズというフィンランドの天才大学生が発明したOS

言語・環境
HTML （エイチティーエムエル、HyperText Markup Language）は、ウェブページを作成するために開発された言語
CSS カスケーティングスタイルシート。文書の見栄えを表現する為のもの。
MySQL 世界で最も利用されているデータベース管理システム
PHP Web開発でよく使用されるスクリプト言語。サーバーサイド言語
JavaScript Webサイトを表示するためにブラウザ上で動くプログラミング言語
jQuery JavaScriptで作成されたライブラリ
Apache オープンソースのWebサーバーソフトウェア
Bootstrap CSSのフレームワーク

エンジニアなど
SE システムエンジニア クライアントの要望・要求に基づいて、システム設計する
PG プログラマー。SEからの仕様書をもとにプログラミングする
Webデザイナー 企業や個人などのクライアントから依頼されたWebサイトのデザインを担当する

ユーザーインターフェース
CUI キャラクターユーザーインターフェース 主に文字でPCとやり取りする
GUI グラフィカルユーザーインタフェース グラフィックや図表でPCとやり取りする

ファイルとディレクトリ
ファイル コンピューターでデータを扱いやすくするためのかたまり
ディレクトリ CUIではディレクトリ、GUIではフォルダ
ルート・ディレクトリ 階層型における最上階層のディレクトリのこと
カレント・ディレクトリ 階層による現在地のこと
パス 経路
絶対パス ルートディレクトリから目的地までの経路
相対パス 現在地からの目的地への経路

各種プロトコル
HTTP 通信規約
FTP (File Transfer Protocol)はサーバーとクライアント間で、ファイルを送受信する通信の決まりごと
HTTPS 通信内容を暗号化している 
SSL Secure Sockets Layer インターネット上でデータを暗号化して送受信する仕組み ユーザーを保護
SSH Secure Shell ネットワークに接続された機器を遠隔操作して、管理するための手段 サーバー管理者を保護
ポート、ポート番号 ポートはインターネットサーバ側の出入り口 そのための出入り口の番号
HTTP、FTP、SSHの各ポート番号 役割によってポート番号が異なる
ファイアウォール 企業などの内部ネットワークをインターネットを通して侵入してくる不正なアクセスから守るための“防火壁”です
TCP Transmission Control Protocol。IPの上位プロトコルのトランスポート層で動作するプロトコル

IPアドレス
IPアドレス インターネット上での住所。ネットワーク上の機器に割り振られている番号。
グローバルIPアドレス 全世界に通用するアドレスのこと
プライベートIPアドレス 特定のネットワークの範囲内で用いられるIPアドレス。

ドメイン
ドメイン インターネット上に存在するコンピュターやネットワークを識別するための名前
DNS (ドメインネームシステム) 主にホスト名に対するIPアドレスを教えてくれるシステム 簡単に言えば辞書
VPN (Virtual Private Network)=仮想専用回線 特定の人のみが利用できる専用ネットワーク
接続したい拠点に専用のルーターを設置し、相互通信を行うことができる
ブリッジ 橋渡し アドレステーブルに登録されたMACアドレスを参照し、送信先が接続されているポートのみにデータを転送する
プロキシサーバー 内部のネットワークとインターネットの境界で動作し、両者間のアクセスを代理して行うもの
電子メール パソコンや携帯電話、スマートフォンなどの情報機器同士が、専用のメールソフトを使って、インターネットなどのネットワークを利用して情報をやりとりする機能
SMTP(Simple Mail Transfer Protocol)送信手順。メールを相手のメールサーバーまで届ける仕組み
POP(Post Office Orieicol) メールソフトが自動的にメールサーバーにアクセスしてメールをダウンロード
IMAP(Internet Message Access Protocol)POPと同じでダウンロードする機能だが、IMAPはPOPと異なりサーバーにメールが残る

セッションとクッキー
セッション セッションとは、通信の開始から終了までのこと
クッキー Webサイトを表示したブラウザのデータを保存しておく仕組み(何度もログインしなくて済む)
キャッシュ ブラウザーで一度表示したページのデータを一時的に保存しておく機能

DB（データベース）について
チューニング 情報システムやコンピュータ、ソフトウェアなどの設定や構成を調整し、目標の状態に近づけたり、性能を最大限引き出したりする作業を指す
スロークエリ データベースにおいて、時間のかかっているSQL(遅いSQL)のことをスロークエリと呼びます。 基本的には、MySQLやPostgreSQLの設定により取得することができ、どの程度遅いものを対象とするか等を指定することが出来る
インデックス インデックスという索引のようなものを使ってデータの検索を高速化する機能

テストについて
フロー 流れのこと
単体テスト モジュールテスト。プログラムの部品がしっかりと機能していることを確認
結合テスト 単体でのテストをクリアしたモジュールと、その他外部モジュールを結合した状態でのテスト

チケット管理ツール
チケット管理ツール チケットを管理するツール。Backlog
タスク ソフトウェアの実行単位のこと

バージョン管理
Git バージョン管理ツール
SourceTree バージョン管理ツール
SVN 集中型バージョン管理システム。
コンフリクト 該当ファイルを複数人で修正してしまうこと。競合
デグレ デグレーション。以前あった物より状態が悪くなってしまうこと
マージツール WinMerge。FDLファイル又はDTDファイルの変更内容をMDLファイルにマージ

その他
ロジック 論理。筋道の通った考え方
デバッグ バグを修正
開発手法(ウォーターフォール開発、アジャイル開発) 開発の時に使われる開発手法
アルゴリズム 手順や計算方法。代表的なものは二分探索方(ニブンタンサクホウ)
フローチャート図 プロセスの流れを箱と矢印を使って表現した図のこと
設計書(外部設計、内部設計) システムをどのように作るか書かれた資料
要件定義(要求仕様書) エンドユーザーが開発担当側に対して開発を依頼するシステム要件を文章化したものです。 こういったことがしたい、という要求と予算、開発体制、リリース時期などの記載
システム開発におけるテスト(単体テスト、結合テスト、総合テスト)その順番に実行していく。
テスト仕様書(どのようなもの、項目) 機能をテストするための切り口をまとめます。これをテスト観点と呼びます。検索機能であれば、「正しくデータを取得できているか確認する」「検索結果が0だった場合の挙動を確認する」などがテスト観点
バリデーション 正規に表現んできているか確認する
FW（フレームワーク）プログラムとかのひな壇のこと
PHPのFW（フレームワーク）の種類と特徴 Laravel,cakePHPなど。プログラムを素早く、簡単に実装できる
Ajax Asynchronous JavaScript + XML JavaScriptとxmlを使って非同期にサーバとの間の通信を行う
API Application Programming Interface APIとはソフトウェアやアプリケーションなどの一部を外部に向けて公開することにより、第三者が開発したソフトウェアと機能を共有
LAMP オープンソースソフトウェアの組み合わせを指す言葉（略称）
クラウドサーバー、レンタルサーバー、VPSの違い CPUやメモリの性能を共有するか否か
AWS Amazonが提供しているコンピューティング、ストレージ、データベース
プラットフォーム サービスやシステム、ソフトウェアを提供・カスタマイズ・運営するために必要な「共通の土台（基盤）となる標準環境」を指す
カバレッジ カバーしている範囲。網羅率。
エビデンス 証拠や根拠
コンパイル プログラミング言語で記述されたソフトウェアの設計図（ソースコード）を、コンピュータが実行可能な形式（オブジェクトコード）に変換する作業のこと
バッチ処理 バッチ単位で行われる処理。PCでひとまとまりのデータを一括して処理する方式
デプロイ(意味と方法、FTPソフトの種類) 配置、展開。開発環境からステージング環境へシステムを反映させることも、ステージング環境から本番環境へ反映させる
FFFTP、Cyberduck
データベース接続クライアント(説明、代表的なもの) データベース管理システム (DBMS) や関係データベース管理システム (RDBMS) 上のデータを容易に扱うためのソフトウェア SQL Workbench/J
Gitホスティングサービス(説明、代表的なもの) Gitによるクラウドベースのバージョン管理サービス 全体統括用のリモートリポジトリと開発者毎のローカルリポジトリを分散してオフラインでも管理ができる
チケット管理ツール(説明、代表的なもの) BackLog,Redmine。プロジェクト毎にタスクを発行して、製作者に対して仕事を依頼するためのもの
DBのJOINとは テーブルの結合のこと
SEO Serch Engine Optimization 検索エンジン最適化。適性に検索エンジンにページ内容を理解・評価されるよう技術的にWebページを最適化すること。
